# Backcall - Обратный звонок (модифицированная версия)

Оригинальный модуль скачан с [sitogon.ru](https://sitogon.ru/joomla-components/obratnyj-zvonok) и модифицирован.

## Описание

Компонент + модуль, позволяющие организовать на сайте прием обратных звонков.

Совместимость: Joomla 5, Joomla 6.

Модуль позволяет отобразить на сайте как кнопку "Заказать звонок", так и форму обратной связи.

В случае отображения кнопки обратного звонка, нажатие на нее открывает окно, где посетитель заполняет имя и телефон. После отправки формы владелец сайта может получить уведомление о заказе звонка как на электронную почту, так и в Telegram.

Все заказы звонков пишутся в базу данных компонента и доступны в административной панели компонента "Обратный звонок".

В случае отображения формы обратной связи, посетитель увидит форму с полями: имя, телефон, email, текст сообщения. Есть возможность прикрепить файл. Все сообщения пишутся в базу данных и доступны в административной панели сайта.

Поддерживается интеграция с Max ботами -- отправка уведомлений об обратных звонках и сообщениях из формы обратной связи в мессенджер Max.

## Установка

Скачанный архив необходимо распаковать. В архиве:

- com_backcall.zip -- компонент
- mod_backcall.zip -- модуль

Устанавливаются через Joomla: Система -> Установка расширений.

## Внесенные изменения

Относительно оригинальной версии 1.0.5 изменены следующие файлы:

- `modules/mod_backcall/tmpl/default_call.php` -- кастомный дизайн модального окна (своя шапка с заголовком и кнопкой закрытия, стилизация под дизайн сайта)
- `administrator/language/ru-RU/com_backcall.ini` -- языковой файл админки
- `administrator/language/ru-RU/com_backcall.sys.ini` -- системный языковой файл
- `language/ru-RU/mod_backcall.ini` -- языковой файл модуля
